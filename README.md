# C++ Terminal Grid Game

Simple terminal game to gain experience writting in cpp.

## How to Play

Move the Player (`P`) to the Goal (`X`) to increase your score.

### Controls
| Key | Action |
| :---: | :--- |
| **W** | Move Up |
| **A** | Move Left |
| **S** | Move Down |
| **D** | Move Right |
| **Q** | Quit Game |

## Getting Started

### Prerequisites
You need a C++ compiler installed (e.g., `g++` or `clang`).

### Build and Run
Open your terminal in the project directory and run:

```bash
# Compile the game
g++ main.cpp -o game

# Run the executable
./game
